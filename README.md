# ToDo
ToDo is a ToDo-App build with React Native, but at the moment, it only works for Android. iOS version comming soon.

## App Features
- **Languages:** German
- **LightMode**
- **DarkMode**
- **AmoledMode**
- **Task to do**
- **Done Tasks**
